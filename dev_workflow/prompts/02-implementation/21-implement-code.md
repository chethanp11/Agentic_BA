# Node: implement-code

## Purpose

Apply the scoped source changes.

## Read

- `plan/code-update.md`
- relevant design artifacts
- current `src/` files

## Write

- `src/` files
- `src/docs/` files when behavior changes

## Output

- source changes that match the plan
- any needed supporting documentation updates

## Rules

- keep changes deterministic
- update documentation when implemented behavior changes
- do not add hidden behavior outside the design
